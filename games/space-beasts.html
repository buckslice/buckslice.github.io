Return of the Space Beasts

Overview

This is a 2D platform game where you shoot space beasts and try to find your spaceship at the end of each level. Made for Windows and Andorid. All the programming, sound effects, and EVEN THE ART was done by myself, and my friend made the intro music for me. I made this game over the last half of the semester for the final project in Software Engineering with UML at SBCC (in 2014). I learned a lot about java and libgdx and got the game working on both windows and android.
